# Development Documentation

This folder contains structured documentation optimized for agentic LLM development.

## Folder Index

| Folder | Purpose |
|--------|---------|
| `01-product/` | Product vision, goals, user stories |
| `02-research/` | Engine analysis (Desmos, GeoGebra, SymPy), technology comparisons |
| `03-architecture/` | System architecture, data flows, protocol design |
| `04-adrs/` | Architecture Decision Records — why we chose what we chose |
| `05-specs/` | Feature specifications and API contracts |
| `06-testing/` | Test plans, verification strategies, showcase equations |
| `07-roadmap/` | Phase planning and milestone tracking |

## How to Use

1. **Before starting work**: Read `../PROJECT_STATE.md`
2. **During work**: Reference relevant docs in this folder
3. **After work**: Update `../PROJECT_STATE.md` with what changed
